Format of input.txt should be like this ;

3
Doe, John
English 90 80 70
Smith, Jane
Science 80 90 85 90
Lee, Peter
Math 80 90 80 70 60 85 70 65

Each line represents a student. The first line specifies the number of students in the file. 
The following lines contain the student's name on the first line, followed by their course and grades on the second line. 
The grades are separated by spaces. The name is in the format lastName, firstName.
